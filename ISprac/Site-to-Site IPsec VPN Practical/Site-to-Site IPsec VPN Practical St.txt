Site-to-Site IPsec VPN Practical Steps
On Router R1
Step 1 — Enter privileged mode

Type

enable

Password

cisco123
Step 2 — Enter configuration mode

Type

configure terminal
Step 3 — Configure ISAKMP policy

Type

crypto isakmp policy 10
encryption aes
hash sha
authentication pre-share
group 2
lifetime 86400
exit
Step 4 — Configure pre-shared key

Type

crypto isakmp key cisco address 10.2.2.2
Step 5 — Configure interesting traffic ACL

Type

access-list 110 permit ip 192.168.1.0 0.0.0.255 192.168.3.0 0.0.0.255
Step 6 — Configure transform set

Type

crypto ipsec transform-set VPN-SET esp-aes esp-sha-hmac
exit
Step 7 — Create crypto map

Type

crypto map VPN-MAP 10 ipsec-isakmp
set peer 10.2.2.2
set transform-set VPN-SET
match address 110
exit
Step 8 — Apply crypto map to interface

Type

interface s0/1/0
crypto map VPN-MAP
exit
end

Verification

Go to PC-A → Command Prompt

Type

Then you will see

This means IPsec VPN tunnel established successfully.
